<Meta title="Element queries/Circular dependencies" />

One of the main reasons we can't have element queries yet, is because **we haven't figured out a way to deal with circular dependencies**.

To implement element queries, the CSS language itself needs to change. It's been several years since element queries were first suggested, and I personally think we won't see them standardised any time soon.
